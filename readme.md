
# EcoLisis-Website

The official website for **EcoPolis App** by EcoLisis - Sustainable Solutions. EcoPolis is a sustainability-focused app that bridges the gap between municipalities, businesses, and citizens, rewarding eco-friendly actions and promoting community engagement for a greener future.

EcoPolis App aims to:

- Support stray animal care by mapping feeding stations and adoption opportunities.
- Enable item exchange and donation within the community.
